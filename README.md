# 🦈 Shark Strike V2

**Shark Strike** is a high-octane 8-bit rhythm game built with **Vue.js 3** and **AWS Amplify Gen 2**. Test your reflexes by matching arrow sequences as fast as you can to climb the global leaderboard!

## 🎮 Features

- **Retro Aesthetic**: Powered by NES.css for a classic gaming feel.
- **Real-time Leaderboard**: Your scores are synced instantly to the cloud using AWS AppSync and DynamoDB.
- **Secure Profiles**: AWS Cognito integration ensures your high scores are yours and yours alone.
- **Fast Performance**: Built with Vite for a smooth, lag-free experience.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or later)
- AWS Account (for cloud features)

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```

## 🛠 Tech Stack

- **Frontend**: Vue.js 3, Pinia, Sass
- **UI System**: NES.css
- **Backend / Cloud**: AWS Amplify Gen 2, Cognito, AppSync, DynamoDB
- **Build Tool**: Vite
